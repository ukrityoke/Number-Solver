🔢 Number Solver (โปรแกรมหาสูตรคำนวณตัวเลข)
โปรแกรมสำหรับค้นหาสูตรและลำดับการคำนวณทางคณิตศาสตร์จากชุดตัวเลข 4-5 ตัว เพื่อให้ได้ผลลัพธ์ตามที่กำหนด (คล้ายเกมถอดรหัสตัวเลข หรือเกม 24) พร้อมแสดงวิธีทำทีละขั้นตอนอย่างละเอียด พัฒนาด้วยภาษา Python และมีหน้าจออินเตอร์เฟส (GUI) ที่ใช้งานง่าย

✨ คุณสมบัติเด่น (Features)
รองรับเครื่องหมายหลากหลาย: * เครื่องหมายหลัก: บวก (+), ลบ (-), คูณ (*), หาร (/), ยกกำลัง (^), แฟกทอเรียล (!), และมอดูโลหรือเศษจากการหาร (%)

เครื่องหมายเสริม: หารปัดเศษลง (//)

วงเล็บ (()): ระบบจะคำนวณและจัดลำดับการใส่วงเล็บให้โดยอัตโนมัติเพื่อให้ได้ผลลัพธ์ที่ถูกต้อง

ความแม่นยำสูง (100% Accuracy): ใช้โมดูล Fraction (เศษส่วน) ในการคำนวณตลอดทั้งกระบวนการ ทำให้หมดปัญหาเรื่องความคลาดเคลื่อนของทศนิยม (Floating-point error)

ค้นหาได้รวดเร็ว (High Performance): * ใช้เทคนิค Memoization เพื่อจดจำสถานะที่เคยคำนวณไปแล้ว ช่วยตัดกิ่งโครงสร้างต้นไม้ (Pruning) ที่ซ้ำซ้อน

มีระบบ Early Stopping ที่จะหยุดการค้นหาทันทีเมื่อเจอคำตอบแรก ทำให้ไม่ต้องรอนาน

ระบบเงื่อนไขขั้นสูง (Advanced Filters):

การใช้เลขซ้ำ: สามารถเลือกได้ว่าจะใช้เลขที่กรอกเพียงครั้งเดียว (ตามกฎดั้งเดิม) หรืออนุญาตให้ใช้ตัวเลขซ้ำกันได้

บังคับเครื่องหมาย: สามารถตั้งค่า "บังคับ" ให้ผลลัพธ์ต้องมีการใช้เครื่องหมายที่กำหนดอย่างน้อย 1 ครั้งได้ (เช่น บังคับว่าต้องใช้แฟกทอเรียล ! หรือต้องมีการหารปัดเศษ //)

แสดงวิธีทำละเอียด: เมื่อพบคำตอบ โปรแกรมจะแตกขั้นตอนการคำนวณออกมาเป็นข้อ ๆ (ขั้นที่ 1, ขั้นที่ 2, ...) เพื่อให้ผู้ใช้ตรวจสอบความถูกต้องและทำความเข้าใจได้ง่าย

🛠️ วิธีการติดตั้งและเริ่มใช้งาน
สิ่งที่ต้องมีในเครื่อง (Prerequisites)
Python 3.x

Tkinter (มักจะติดตั้งมาพร้อมกับ Python อยู่แล้วบน Windows/macOS)

ขั้นตอนการรันโปรแกรม
ดาวน์โหลดหรือคัดลอกไฟล์ number_solver.py ไว้ในเครื่องของคุณ

เปิด Terminal หรือ Command Prompt แล้วเข้าไปที่โฟลเดอร์ที่เก็บไฟล์

รันคำสั่งต่อไปนี้เพื่อเปิดโปรแกรม:

Bash
python number_solver.py
💡 วิธีการใช้งานหน้าโปรแกรม
เลือกจำนวนตัวเลข: ติ๊กระบุระหว่าง 4 ตัวเลข หรือ 5 ตัวเลข

กรอกตัวเลข: ใส่ตัวเลขที่ต้องการนำมาคำนวณในช่อง (รองรับทั้งจำนวนเต็ม, ทศนิยม, หรือเศษส่วน เช่น 3/4)

กำหนดคำตอบ: กรอกผลลัพธ์สุดท้ายที่ต้องการในช่อง "คำตอบที่ต้องการ"

ตั้งค่าเครื่องหมาย & เงื่อนไข:

ช่อง อนุญาต: ติ๊กเลือกเครื่องหมายคณิตศาสตร์ที่ยอมให้ระบบนำไปใช้

ช่อง บังคับต้องมี: ติ๊กเลือกหากต้องการให้สมการนั้น "จำเป็น" ต้องใช้เครื่องหมายดังกล่าว

ติ๊กเปิด/ปิด "อนุญาตให้ใช้ตัวเลขซ้ำได้" ตามความต้องการ

เริ่มคำนวณ: กดปุ่ม 🔢 คำนวณ (Calculate) เพื่อเริ่มค้นหา (หากใช้เวลานานเกินไป สามารถกดปุ่ม หยุด ได้ตลอดเวลา)
