# ตัวช่วยหาสูตรคำนวณตัวเลข

หาสูตรคำนวณจากตัวเลข 4-5 ตัว ให้ได้คำตอบตามที่กำหนด โดยใช้ `+ - * / ^ ! %` และวงเล็บ

## ไฟล์ในชุดนี้
- **`number_solver.py`** — เวอร์ชัน Desktop (Python + tkinter) ใช้บน Windows / Mac / Linux
- **`number_solver.html`** — เวอร์ชันเว็บ ไฟล์เดียว ใช้บน iPad / iPhone / เบราว์เซอร์ทั่วไป (ไม่ต้องติดตั้งอะไร)

## วิธีใช้งาน

**Python:** รันคำสั่ง `python3 number_solver.py` (ต้องมี tkinter ติดตั้งอยู่แล้ว)

**เว็บ (iPad ฯลฯ):** เปิดไฟล์ `number_solver.html` ด้วย Safari/Chrome ได้เลย หรือกด "เพิ่มไปยังหน้าจอโฮม" เพื่อใช้เหมือนแอป

## ฟีเจอร์หลัก
1. เลือกใช้ 4 หรือ 5 ตัวเลข
2. เลือกได้ว่าจะ **อนุญาต** เครื่องหมายไหนบ้าง (ค่าเริ่มต้นเปิดหมด ยกเว้น `%`)
3. เลือกได้ว่าจะ **บังคับต้องมี** เครื่องหมายไหนในคำตอบ (เช่น บังคับต้องมี `!`)
4. เลือกได้ว่าจะอนุญาตให้ใช้ **ตัวเลขซ้ำ** หรือไม่ (ค่าเริ่มต้น: ไม่ซ้ำ)
5. กด "คำนวณ" แล้วระบบจะแสดงสูตรที่ได้ พร้อม **วิธีทำทีละขั้น** ให้ตรวจสอบความถูกต้อง

ทุกการคำนวณใช้เลขคณิตแบบเศษส่วนแม่นยำ (ไม่ใช่ทศนิยม) จึงไม่มีปัญหาความคลาดเคลื่อน

## หมายเหตุเรื่องความเร็ว
เวอร์ชันเว็บอาจช้ากว่าเวอร์ชัน Python ในบางเคส (โจทย์ที่ไม่มีคำตอบ, เปิดเลขซ้ำ, บังคับเครื่องหมายที่หายาก) เพราะข้อจำกัดของ BigInt ใน JavaScript — ถ้าค้นหานานเกิน 15 วินาทีในเวอร์ชันเว็บ ระบบจะหยุดและแจ้งว่าไม่พบคำตอบโดยอัตโนมัติ
